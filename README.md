

[This is a fork](https://github.com/Yellow-Dog-Man/mediawiki-extensions-Mpdf) of the [fork of](https://github.com/GSTbyMM/Mpdf) the [original extension Mpdf](https://github.com/wikimedia/mediawiki-extensions-Mpdf) working with mpdf 8.2.5 and MediaWiki 1.43.1.


It was re-forked from the [original](https://github.com/Yellow-Dog-Man/mediawiki-extensions-Mpdf) in order to re-apply [GSTbyMM](https://github.com/GSTbyMM/Mpdf)'s work but also stay in sync with upstream changes in a change trackable way. 

We also wanted to ensure that the changes required to work with MPDF 8.x etc were as minimal as possible.


## Installation & Usage

Refer to the [original extension's MediaWiki Page](https://www.mediawiki.org/wiki/Extension:Mpdf), but replace the git url with this repositories one: `https://github.com/Yellow-Dog-Man/mediawiki-extensions-Mpdf.git`


## Upstream

It's highly likely that the original extension, will get similar updates shortly at which point this fork will be no longer needed.
